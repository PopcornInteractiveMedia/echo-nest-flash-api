createLoader()method 
public function createLoader(options:Object, responseProcessor:Function, ... responseProcessorArgs):URLLoader

Creates a loader with event listeners.

The following options are supported:

OptionEvent
onComplete Event.COMPLETE
onResponse Called with the processed response.
onEchoNestError Called with an EchoNestError if the status code is nonzero
onProgress ProgressEvent.PROGRESS
onSecurityError SecurityErrorEvent.SECURITY_ERROR
onIoError IOErrorEvent.IO_ERROR
onError IOErrorEvent.IO_ERROR
SecurityErrorEvent.SECURITY_ERROR
onHttpStatus HTTPStatusEvent.HTTP_STATUS
Parameters
options:Object — The event listener options for the loader.
 
responseProcessor:Function — The function that processes the XML response.
 
... responseProcessorArgs — The arguments to pass to the response processor function.

Returns
